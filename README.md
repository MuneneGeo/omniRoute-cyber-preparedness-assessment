## Key Areas Covered

- Adversary classification using the Cyber Preparedness adversary class framework
- MITRE ATT&CK TTP mapping — 14 techniques across 7 tactic categories
- Goal-to-consequence mapping — financial, operational, legal, reputational
- Cyber Attack Lifecycle defensive mapping — Recon through to Maintain
- Sysmon detection engineering — EID 1 (OriginalFileName), EID 6, EID 11, EID 12/13
- Multi-jurisdictional regulatory analysis — GDPR, Kenya DPA, Singapore PDPA
- 14-dimension preparedness gap analysis across Governance, Operations, and Architecture
- Preparedness posture recommendation — Architectural Resilience
